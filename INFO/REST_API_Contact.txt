Отримання всіх даних
GET
http://localhost:8080/api/v1/contacts

------------------------------------------------------
------------------------------------------------------

Отримання даних за id
GET
http://localhost:8080/api/v1/contacts/1

------------------------------------------------------
------------------------------------------------------

Отримання даних за name
GET
http://localhost:8080/api/v1/contacts/query-by-name?name=Leo

------------------------------------------------------
------------------------------------------------------

Створення даних
POST
http://localhost:8080/api/v1/contacts

Body приклад 1:
{
    "name": "Leo",
    "phone": "000 000-0000"
}

Body приклад 2:
{
    "name": "Nick",
    "phone": "120 081-8888"
}

Якщо невалідні дані (в даному прикладі телефон має неправильний формат):
{
    "name": "Nick",
    "phone": "120 81-88"
}

Відповід від сервера:
{
    "status": 400,
    "message": "Bad Request",
    "success": false,
    "data": BAD_REQUEST
}

------------------------------------------------------
------------------------------------------------------

Оновлення даних
PUT
http://localhost:8080/api/v1/contacts/1

Body:

    "phone": "134 456-7890"
}

------------------------------------------------------

http://localhost:8080/api/v1/contacts/1
